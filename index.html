Pengenalan

Borang Maklumat Murid untuk sekolah di Malaysia.

Ini adalah borang ringkas yang boleh ibubapa isi dan cetak kemudian untuk diberikan ke pihak sekolah.

Saya Iszuddin Ismail. Saya membina borang ini bagi mengelak keperluan untuk menulis tangan borang ini berkali-kali bagi setiap anak yang bersekolah. Malah dengan ini anda boleh menyimpan maklumat yang telah dimasukkan untuk kegunaan untuk anak lain, dan juga untuk tahun berikutnya.

Ini sebenarnya projek peribadi saya sebab penat nak isi borang ni setiap tahun untuk semua anak. Pastinya borang ini berbeza mengikut sekolah. Terpulanglah nak guna macam mana.

Kawan-kawan programmer boleh fork atau download projek ni di GitHub kalau nak di https://github.com/kidino/borang-sekolah

Storan Data

Data yang ditaip di sini TIDAK DISIMPAN DI SERVER. Ianya menggunakan storan LocalStorage di pelayar (browser) masing-masing. Maka saya tidak menyimpan maklumat anda.

Cetakan

Apabila mencetak, gunakan setting kertas A4, dan custom margin. Saya cadangkan margin berikut:

Atas: 0.26"
Bawah: 0.65"
Kanan: 0.38"
Kiri: 0.43"

Dicadangkan buangkan setting Header & Footer

Ujian cetakan hanya dijalankan menggunakan pelayar Chrome.

Pepijat & Ralat

Ini projek sampingan peribadi dibangunkan dalam sehari dua. Pastinya ada pepijat dan ralat. Saya tak jamin saya akan baiki apa-apa masalah tersebut.
